# Paleo-Coordinate reconstructions for Ocean Drilling Sites

Tool for reconstructing paleo-lat/lon using pyGPlates.

## Structure
.
├── extract_paleo_coordinates.py   # Main processing script
├── test_dataset/                  # Input CSV files
│   └── Template_Site982.csv
└── GPlates_features/              # Tectonic model files
    ├── Rotations/                 # .rot files
    └── StaticPolygons/            # .gpmlz or .shp files

## Required Python Packages
•	pygplates
•	pandas
•	numpy
For detailed documentation, including installation instructions for pygplates, visit the official site:  https://www.gplates.org/docs/pygplates/


## Input Data Format
your test_dataset/*.csv files should strictly follow this structure:
Your files in test_dataset/*.csv must strictly follow this structure:
Column Name	Description	Example Data
Site	Ocean drilling site name	982
Latitude	Decimal degrees (-90 to 90)	56.68
Longitude	Decimal degrees (-180 to 180)	-23.59
Age (Ma)	Reconstruction time (Millions of years)	10.5

## How to Use
1.	Place your site data CSV in test_dataset/.
2.	Ensure your rotation (.rot) and polygon (.gpmlz) files are in the GPlates_features/ subfolders.
3.	Run the script from your terminal:
Bash
python extract_paleo_coordinates.py
