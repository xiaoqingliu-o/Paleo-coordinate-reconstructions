# Paleo-Coordinate Reconstructions

A tool for reconstructing paleo-latitude and paleo-longitude using
**pyGPlates**.

------------------------------------------------------------------------

## Repository Structure

    .
    ├── extract_paleo_coordinates.py   # Main processing script
    ├── test_dataset/                  # Input CSV files
    │   └── Template_Site982.csv
    └── Plate_model_rotation_and_StaticPolygons_files/              # plate model files
        ├── muller2008/                 # muller2008 plate model
        └── zahirovic2022/            # zahirovic2022 plate model

## Input Data Format

CSV files in `test_dataset/` **must follow** this structure:
- First column: site name  
- Second column: modern longitude
- Third column: modern latitidue
- Fourth column:  Reconstruction time (millions of years)

| Site | Longitude | Latitude | Age (Ma) |

## Plate Model References

If you use results generated by this tool in a publication, please cite
the associated reference:

-   Zahirovic et al. (2022)\
    *Subduction kinematics and carbonate platform interactions.*\
    Geoscience Data Journal.\
    https://doi.org/10.1002/gdj3.146\


-   Müller et al. (2008)\
    *Age, spreading rates, and spreading asymmetry of the world's ocean crust.*\
    Geochem. Geophys. Geosyst.\
    https://doi.org/10.1029/2007GC001743\

Notes: Most published Miocene (MioMIP1) and Eocene (DeepMIP-Eocene) simulations use paleogeography based on the plate kinematic model of Müller et al. (2008). For pointwise model–data comparisons using these simulations, reconstructing paleo-coordinates with the Müller et al. (2008) plate model is recommended.

For other applications, the plate model of Zahirovic et al. (2022) may be preferred. As the DeepMIP community is planning future MIP simulations based on paleogeography from Zahirovic et al. (2022), model–data comparisons using those simulations should use the Zahirovic et al. (2022) plate model for consistency.

## How to Use

1.  Place your site data CSV file in `test_dataset/`.\
2.  Ensure rotation (`.rot`) and polygon (`.gpmlz` or `.shp`) files are
    located in the appropriate subfolders under `GPlates_features/`.\
3.  Run the script from your terminal:

``` bash
python extract_paleo_coordinates.py
```
